# Amazon Price Monitor

Projeto em Python que monitora preços de produt**New!Click to edit**os da Amazon automaticamente.

## Funcionalidades

* Coleta preço real de produtos
* Salva histórico em CSV
* Alerta quando preço abaixa
* Estrutura modular

## Tecnologias

* Python
* Selenium

## Como usar

1. Instale dependências:

```bash
pip install -r requirements.txt
```

2. Configure o produto no `config.py`

3. Execute:

```bash
python src/bot.py
```

## Saída

* Histórico salvo em `data/historico.csv`
* Alertas no terminal

## Objetivo

Projeto para demonstrar automação e web scraping com Python.
