Amazon Price Monitor

Projeto em Python que monitora preços de produtNew!Click to editos da Amazon automaticamente.

Funcionalidades
Coleta preço real de produtos
Salva histórico em CSV
Alerta quando preço abaixa
Estrutura modular
Tecnologias
Python
Selenium
Como usar
Instale dependências:
pip install -r requirements.txt
Configure o produto no config.py
Execute:
python src/bot.py
Saída
Histórico salvo em data/historico.csv
Alertas no terminal
Objetivo

Projeto para demonstrar automação e web scraping com Python.
